===============================================================================
はじめに
===============================================================================
ダウンロードしていただきありがとうございます。
このキャラはルージュノワール氏によって制作された「遠野志貴」を七夜風に改変した
キャラです。他の改変志貴のような、殺人貴仕様ではなく、ほぼ原作七夜仕様ですので、
火力はかなり落ちています。
・・・と言っていましたが、コンボ火力は最近になって半端ないことが分かりましたｗ

===============================================================================
使用可能ＭＵＧＥＮ
===============================================================================
WinMugen専用と一番上に書いてありますが、MugenRCでも使用できます。ただし、DosMugen
では使用できません。

===============================================================================
操作方法
===============================================================================
操作方法はCommandList.txt を見てください。

===============================================================================
裏七夜モード
===============================================================================
７Ｐ～１２Ｐを選ぶと、必殺技などが変化した裏七夜モードが使えます。
操作方法はCommandList-2.txt を見てください。

ＡＩについて
===============================================================================
AI ReadMe by 歩き明星氏.txtを読んでください。
