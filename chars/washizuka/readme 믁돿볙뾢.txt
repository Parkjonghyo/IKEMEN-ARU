最初に

ほぼ、オリジナルの月華2に近い性能にしてあります。
最初に公開したキャラが、改変強化キャラ（いわゆるマスターキャラ）であったので、
通常のキャラを相手にするには強いという問題がありますので、
オリジナルに近い通常版の作成とそのAIを公開しようと思い作成しました。

マスター版のような、各種昇華、コマンド投げ、ゲージの自動増加、隠し剣質といった要素は搭載されず、
キャラ性能的には比較的平凡となっています。
（オリジナルが元から連続技二回でKOするキャラなので、攻撃力は強めですが(;´Д｀)）

また、マスター版は、ゲージの自動増加やパワーゲ−ジがLv6であることを生かした立ち回りをすることが想定されていたため、
本来の月華オリジナルの仕様であるタイプBにあたる、
パワーゲージがLv.1までしか溜まらないかわりに、ライフが1/4(250以下)になった場合に、
・超必殺技がゲージに関係なく出せる。
・潜在奥義・乱舞奥義がパワーゲージLv.1で出せる。
といった、仕様できちんと立ち回れるAIは作っていませんでした（選択自体は可能ですが）。

そこで、この通常版には、本来の月華のゲージシステムであるタイプBに対応したAIも製作しました。

＊ちなみに、A〜Zボタンで決定するパレット1-6が、ゲージ３本性のタイプA
　スタートボタンを押しながら、A〜Zボタンで決定するパレット7-12が、ゲージ１本＋ライフ性のタイプBです。

オリジナルとの相違点


1.ガードキャンセルの挙動

412+y　で、ゲージ1消費でガードキャンセルが出ます。

ヒットした時の挙動はオリジナルとは異なり、マスター版と同様にしてあります。（かためキャラ対策です。）
地上の相手に当てたときはノックバック、空中の相手に当てた時は空中に打ち上げます。
また、必殺技でキャンセル可能にしてあります。

このガードキャンセル攻撃の強化が、ほぼ唯一のオリジナルに対する強化点といえます。


2.超必→潜在の擬似昇華を可能に

真狼牙→最終狼牙の擬似昇華が可能です。
これは通常の仕様では最終狼牙の確定状況は一切なく、
「誠」を味わうことが非常に困難ですので、どうしても「誠」の旗を拝むための苦肉之策です。

真狼牙の俊殺連のモーション時に最終狼牙のコマンドを入れれば、ゲージ0.5本分を消費して昇華できます。

ただし、これは演出を味わうための追加ですので、実質のダメージはゲージを使っているのにほとんど増えません。
マスター版のように真狼牙を打ち切ってからではなく、真狼牙を途中で止めるうえに、
追撃用にダメージを調整された弱めの最終狼牙を継ぎますので、
真狼牙を最後まで入れるのと比べてダメージはやや高いだけで、
真狼牙の後にきっちり俊殺で追撃すれば、ダメージはほとんど一緒ですｗ

ですので、これは強化目的とかではなく、ただ純粋に「誠」の旗を拝むための、演出目的の個人的な趣味です( ﾟДﾟ)


3.一部追加技あり

「DALA」さんが作成されているオリジナル技の「狼牙・上式」と「疾空狼牙」はリスペクトもかねて残してあります。

上式に関しては、こちらは昇華不可ですが、虚空殺や小斬りなどがはいりますので、
ゲージのあるときは昇華を狙って 直式→真・狼牙 を、ない時は 上式→虚空殺 と使い分けるといいと思います。

疾空狼牙は昇華はできませんが、真・狼牙よりダメージ自体は高いので、お好みで連続技に組み込むか、
遠距離からのぶっぱにお使いください。



＊立ち回りの自由度などを考えると、コマンド投げ程度はこちらにもあってもよかったかもしれません。
　ご要望があれば、スイッチ式にして部分的な昇華やコマンド投げのon/offを可能にしても良いかなと思います。

AIについて

AIは11段階のレベル式AIです。
AI.cns　の中の

[State -3, AI]
Type = VarSet
TriggerAll = ((Var(59) = 0) && (RoundState != 3))
trigger1 = command = "AI_01"
　　　　　〜
　　　　　中略
　　　　　〜
trigger50 = command = "AI_50"
Trigger51 = Numhelper(10000)
Trigger51 = helper(10000),var(59)
V = 59
Value =11 ; 1〜11でレベルを決めてください。1が弱く、11が最強です。
IgnoreHitPause = 1

この、Value =　の後に数字1〜11を入れてください。
レベル1が最弱で、11が最強です。

連続技

基本的な流れは、昇華可能剣質なら

・通常技→狼牙→直式→真狼牙→最終狼牙（追撃）（ゲージがないなら弱俊殺）
・通常技→狼牙→上式→虚空殺

の2種類が基本です。昇華できない剣質「技」のみは、通常技から直接に真狼牙をつないでください。
また、ゲージさえあれば乱舞奥義の使える剣質なら、（通常技→乱舞奥義→）を頭にはさむ事が出来ます。
狼牙のところは、お好みで俊殺、弱疾空殺、疾空狼牙などをつないでも良いです。
