 初期設定
----------
付属のKamijo_-2.cnsを開きエンハンスの初期設定、開始時のゲージ設定、バーストアイコンの位置設定、ハードラックアイコンの位置設定を変更できます


 基本動作
----------

	-弱攻撃     		a
	-中攻撃     		b
	-強攻撃     		c

	-投げ       		F, x (間合い内)
	-空中投げ   		空中 F, x (間合い内)
	
	-ダッシュ   		F, F
	-空中ダッシュ		空中 F, F
	-バックステップ		B, B

 特殊動作
----------
	-前転or後転 		a+b or B, a+b

	-ゲージ溜め 		a+c
	-くらい抜け 		バースト使用可能時に食らい中にa+b 空中でも使用可
	
	-幻想殺し(弾き)			D, DF, F, x 
	上段の打撃技を弾いて相手を硬直させる
	次の攻撃1.1倍
	超必殺技でキャンセルできる
	
	-幻想殺し(シールド)		x
	-そげぶカウンター		幻想殺し(シールド)成功後 D, DF, F, x 
	飛び道具をシールドしてカウンター時ゲージ消費なし それ以外の攻撃をシールドしてカウンター時 1ゲージ消費
	
	-ハードラックエンハンス		y or x+y	後述
	
	-ゲジマユエンハンス			強運時にy	後述
	-アーマーエンハンス			凶運時にx+y	後述

	-FRC,ロマンキャンセル		z			後述

-| 特殊システム |----------------------------------------------------------------
カウンターアタック
 - 相手が攻撃動作中に当てる事で補正が緩和,一部の技の性能が変化

幻想殺し	
 - そげぶ技中に飛び道具を打ち消せる
 
ハードラックシステム（主人公補正）
 - エンハンスをすることで性能が変わる
=不幸状態=
 - BadLuck 初期状態
	ゲージ技使用不可
	20%の確率で被ダメージ(44.4％)増加
	そげぶ技補正95%
	
 - UnLucky BadLuckからのエンハンス（yボタン）使用後
	1ゲージ技使用可
	10%の確率で被ダメージ(44.4％)増加
	そげぶ技補正97.5%
	喰らい抜け使用不可
	
 - HardLuck(白) UnLuckyからのエンハンス（yボタン）使用後
	3ゲージ技まで使用可
	5%の確率で被ダメージ(44.4％)増加
	そげぶ技補正100%
	強運状態移行可  	y
	凶運状態移行不可
	
 - HardLuck(赤) 一定条件後に自動で移行
	3ゲージ技まで使用可
	10%の確率で被ダメージ(44.4％)増加
	そげぶ技補正105%
	強運状態移行可   y
	凶運状態移行     x+y

 - HardLuck(黒)	強運、凶運エンハンス失敗時
	3ゲージ技まで使用可
	20%の確率で被ダメージ(44.4％)増加
	そげぶ技補正100%
	強運状態移行不可
	凶運状態移行不可
	
=強運状態=	
 - HardLuck(強)	HardLuck(白)(赤)からのエンハンス（yボタン）使用後
	5ゲージ技まで使用可
	被ダメージ増加無し
	そげぶ技補正110%
	FRC使用可
	ゲジマユエンハンス使用可能
	
=凶運状態=
 - Calamity	HardLuck(赤)からのエンハンス（x+yボタン）使用後
	3ゲージ技まで使用可
	竜王の顎使用可
	25%の確率で被ダメージ(66.6％)増加
	そげぶ技補正120%
	飛び道具完全無効
	ガード使用不可
	ゲジマシ
	アーマーエンハンス使用可能

=天運状態=
 - SYSTEM	Kamijo_-2.cnsを開きエンハンスの初期設定を変更することで移行可能
	全ゲージ技使用可
	被ダメージ増加無し
	そげぶ技補正125%
	飛び道具完全無効
	FRC使用可
	ゲジマユエンハンス使用可能
	アーマーエンハンス使用可能

根性値
 - 体力によってダメージ軽減
	BadLuck,UnLucky,HardLuck(黒)時
	体力が30%以下の時30％軽減
	
	HardLuck(白),HardLuck(赤)時
	体力が30%以下の時40％軽減
	
	HardLuck(強),Calamity時
	体力が30%以下の時60％軽減

	SYSTEM時
	体力が30%以下の時70％軽減

ゲジマユエンハンス（強運時のみ）
 - 強運時にyで一定時間ゲージ技の制限がなくなる ゲージ本数に応じて時間が延びる 最低3ゲージが必要
   体力が三割未満のときは GoodLuckの文字が表示され一撃技が使用可能
   特定の技を使うか食らいぬけを使用するとゲージが残っていても終了

アーマーエンハンス（凶運時のみ）
 - 凶運時にx+yで一定時間アーマーを付加する 3ゲージが必要

ロマンキャンセル（強運時以外）
 - 打撃技を相手に当てた時に、打撃技のモーションや硬直をキャンセル出来る（例外あり） 
	打撃技を相手に当てた時に、z （2ゲージ消費）

FRC（強運時のみ）
 - 打撃技を相手に当てた時に、打撃技のモーションや硬直をキャンセル出来る（例外あり） 
	打撃技を相手に当てた時に、z （1ゲージ消費）
