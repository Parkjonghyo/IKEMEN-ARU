●システム
・要撃
相手の通常投げを打撃で、相手の攻性防禦を投げでつぶすと「要撃」となり、
打撃や投げの威力が若干上がります。

・攻性防禦
YまたはB+Cを押すと攻性防禦モーションを取り、この状態で相手の攻撃を受けると自動で反撃を行います。
この反撃は相手を浮かせ、必殺技以上の攻撃でキャンセルする事が可能です。
立ち、屈み、空中で出せますが、立ちはしゃがみ攻撃を、しゃがみは空中攻撃を取れません。
飛び道具を取った場合は自動反撃を行わず、即行動可能になります。
ちなみに、反撃に無敵などは一切ありませんが、攻性防禦を攻性防禦でキャンセル可能ですので、攻撃のヒット数分攻性防禦を行えば反撃が可能です。
また、反撃は出がかりを必殺技以上の技でキャンセルすることができます。
なお、立ちで上段や中段を、しゃがみで下段をといった具合に対応した段で取れば「要撃攻性防禦」となります。
反撃の浮きとダメージが若干増加します。

・体力差補正
相手と自分の体力に応じて攻撃力が変化します。
相手より自分の体力が高ければ低く、自分の体力が低ければ高くなります。

・防禦崩壊
敵の攻撃をガードすると、体力の下にある防禦ゲージが減っていきます。
ゲージが0になるとガードが崩れて大きな隙を晒してしまいます。
防禦ゲージは一律300ポイントで、ガードすると受ける筈のダメージだけ減ります。
屈みガードした場合は減少量に×1.25、空中ガードは×2.00の補正がかかります。
連続ガード中は、ガード2段目に×0.75、3段目に×0.50、4段目以降に×0.25の補正がかかります。
防禦ゲージは時間と共に回復します。

・受け身
相手の攻撃で吹き飛ばされたとき、地面に落ちる直前に何かのボタンを押すと受け身を取ることができます。
受身中は完全無敵です。また、入力は一度しか受け付けません。

・ﾀﾒ成立フラグの持続
ﾀﾒが完了し、技が撃てるようになった状態が少しの間維持されます。
例えば2ﾀﾒ8+攻→8B+Cでフラクトリットキャンセル特攻フラクトリットが出ます。

・完全煉鎖
A→A→A→AやB→B→B、C→Cのようにボタンを順番に押すことで前の動作をキャンセルして別の動作が出ます。
一部煉鎖から別の煉鎖に派生することもできます。

・ダッシュ攻撃
フロントステップ中に攻撃するとダッシュ攻撃が出ます。ヒットすると相手をふっ飛ばし画面端に近い時は壁やられを誘発します。

・壁やられ
壁に張り付いた相手に攻撃をたたきこむチャンス。

・二段ジャンプ
空中でもう一度ジャンプできます。使用後は着地までガード不能なので注意。

・ヴァイタルソース
体力の最大から現在の体力を引いた3分の1がヴァイタルソースとして記憶されます。
発動時はその3分の2が回復（のけぞり発動なら3分の1が）、
以降完全世界中は発動時のヴァイタルソース部分まで徐々に回復します。

・完全世界
A+B+Yを押すと発動します。発動時に相手をふっ飛ばすダメージなしの判定が出ます。のけぞり中でも使用可能。
発動すると体力がヴァイタルソース分だけ回復し、攻撃力が上昇。
一定時間経過か完全神殺を出すと終了します。
発動したラウンドは以降ゲージが一切たまらなくなるので注意。

・完全神殺
完全世界中にA+B+Yを押すと発動します。最終特別攻撃に匹敵する高威力の技。
ヒットするか否か、つぶされたかどうかを問わず使用後は完全世界が終了します。

・特殊カラー
キャラ選択時に特定の色を選択すると、特別な効果が受けられます。

[黒カラー]
特攻ゲージ常時MAX

[金カラー]
攻撃力上昇
防御崩壊なし
特攻ゲージ常時MAX
弾数無限

・KO演出
-2ステートのファイルに
[State -2,KO演出スイッチ]
type = VarSet
trigger1 = 1
var(36) = 1;0にするとＫＯ演出と勝利画面オフに

となっている部分があります。
var(36) = 0とすることでKO時の演出及び勝利画面が出なくなります。
